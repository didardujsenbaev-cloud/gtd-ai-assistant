# Инструкция: Запуск GTD Scripts

## Скрипт 1 — GTD Master System (личный Google)

### Что создаст:
- `📥 INBOX` — точка захвата
- `🗂 PROJECTS` — все проекты
- `⚡ NEXT ACTIONS` — следующие действия
- `⏳ WAITING FOR` — авто-фильтр задач в ожидании
- `💭 SOMEDAY` — когда-нибудь/может быть
- `🎯 AREAS` — 16 областей ответственности (уже заполнено)
- `📊 WEEKLY REVIEW` — трекер еженедельных обзоров

### Шаги:

1. Открыть **личный Google аккаунт** (Master Workspace)
2. Перейти на **drive.google.com**
3. Создать папку `00_GTD_SYSTEM` (если ещё нет)
4. Внутри папки создать новый **Google Sheets** (`+ New → Google Sheets`)
5. В открытом файле перейти: **Extensions → Apps Script**
6. Удалить весь существующий код в редакторе
7. Открыть файл `GTD_Master_System.gs` из этой папки и **скопировать всё содержимое**
8. Вставить в редактор Apps Script
9. Нажать **Save** (иконка дискеты или Ctrl+S)
10. В выпадающем списке функций выбрать **`createGTDMasterSystem`**
11. Нажать **▶ Run**
12. При первом запуске появится запрос на разрешения → нажать **Review permissions → Allow**
13. Дождаться сообщения: `✅ GTD Master System создан!`
14. Вернуться в таблицу — увидишь 7 вкладок

## Скрипт 2 — GTD Biz · Узаконение (рабочий аккаунт)

### Что создаст:
- `👥 КЛИЕНТЫ` — реестр клиентов с финансами
- `🏠 ОБЪЕКТЫ` — объекты недвижимости в работе
- `📋 ЭТАПЫ РАБОТ` — стандартные этапы по каждому объекту
- `📄 ДОКУМЕНТЫ` — реестр документов

### Шаги:

1. Открыть **Google аккаунт Узаконения** (рабочий)
2. Перейти на **drive.google.com**
3. Создать папку `00_CLIENTS` (если ещё нет)
4. Внутри создать новый **Google Sheets**
5. В открытом файле: **Extensions → Apps Script**
6. Удалить существующий код
7. Открыть файл `GTD_Biz_Legalization.gs` и **скопировать всё содержимое**
8. Вставить в редактор
9. Сохранить (Ctrl+S)
10. Выбрать функцию **`createGTDBizLegalization`**
11. Нажать **▶ Run**
12. Выдать разрешения при запросе
13. Дождаться: `✅ GTD Biz · Узаконение создан!`

## После запуска скриптов

### В GTD Master System:
1. Перейти на лист `🎯 AREAS` — проверить 16 областей, добавить свои если нужно
2. Перейти на `🗂 PROJECTS` — удалить строку-пример, добавить первые реальные проекты
3. Для каждого активного клиента (из Узаконения) создать проект:
   - Название: `Узаконить объект [адрес]`
   - Желаемый результат: `Получен акт ввода в эксплуатацию`
   - Область: `Legalization`
   - Бизнес-аккаунт: `Google: Узаконение`
4. Для каждого проекта добавить Next Action в `⚡ NEXT ACTIONS`

### В GTD Biz · Узаконение:
1. Перейти на `👥 КЛИЕНТЫ` — удалить пример, добавить всех реальных клиентов
2. Для каждого клиента добавить объект в `🏠 ОБЪЕКТЫ`
3. Для каждого объекта добавить этапы в `📋 ЭТАПЫ РАБОТ`
4. В поле `ID Проекта (Master)` у каждого клиента вписать ID проекта из GTD Master

### Связь между файлами:
- В `GTD Biz · Узаконение → КЛИЕНТЫ`, колонка `J (ID Проекта Master)` = `PRJ-XXX`
- В `GTD Master → PROJECTS`, колонка `A (ID)` = тот же `PRJ-XXX`
- Это единственная связь — ручная, простая, надёжная

## Если что-то пошло не так

| Проблема | Решение |
|----------|---------|
| Скрипт не запускается | Убедись что выбрана правильная функция в выпадающем списке |
| Ошибка прав доступа | Нажми "Advanced" → "Go to (unsafe)" → Allow |
| Листы создались пустые | Запусти скрипт ещё раз — он очищает и пересоздаёт |
| Формулы не работают | Проверь что названия листов точно совпадают (с эмодзи) |
